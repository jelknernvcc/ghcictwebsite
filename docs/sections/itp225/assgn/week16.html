ITE 170: Linux System Administration

Assignment 16: The Boot Process

Due Date

Friday, May 20th, 2016.

Competencies

Be able to:

  • list system daemons and network services started by the systemd service and socket units.
  • control system daemons and network services using systemctl.
  • describe the boot process.
  • repair file system issues at boot.
  • repair boot loader problems.
  • recover access to a virtual machine.
  • set up an installation server.
  • set up a TFTP and DHCP server or PXE boot.
  • configure DHCP for PXE boot.
  • create a kickstart file.

What to do:

  1. Read chapters 19 and 20 from our text.
  2. Complete all the labs from both chapters.
