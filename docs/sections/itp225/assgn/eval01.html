Performance Evaluation 1: Base Installation

Due Date

Friday, April 15, 2016 at the end of class.

Competencies

Be able to install a Centos 7 system from DVD with specified parameters and options.

Evaluation

Given a laptop and a Centos 7 DVD, do a Server GUI install of Centos on the laptop with the following specifications:

  1. System should have a 500 MiB standard partition which mounts /boot and a 1 GiB swap partition.
  2. System should have LVM partitions of the following sizes, mounts and file systems:
    SizeMountFile System
    30 GiB/xfs
    10 GiB/homeext4
    remaining space/varext4
  3. Networking should be enabled using DHCP with each student setting the hostname of their machine as indicated in the following table:
    StudentMachine Name
    Grahamlaptop1
    Jonathanlaptop2
    Marcolaptop3
    Martinlaptop4
    Tripplaptop5
  4. The root password should be set to password.
  5. There should be a regular (without admin privileges) user named user with password password.
  6. Packages needed for KVM installation should be installed.
