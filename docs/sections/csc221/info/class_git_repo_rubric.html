CSC 221 Class Git Repo Grading Rubric

Grade Description
A

Commits are present in the commit history showing regular engagement with course assignments, both programs and notes. These commits show clear evidendance of an incremental development process.

All notes should be written in Markdown files and should contain evidence of understanding and be presented in a way that demonstrates pride in authorship.

B All required work is present, but it is lacking in either evidence of sufficient engagement or is of insufficient quality to earn an A.
C The minimal required files are present, but the commit history suggests they were not developed with sufficient care or effort, and that were not worked on over an extended time.
D Some work was completed, but it shows only minimal effort and does not provide sufficient evidence of goal attainment or learning mastery.
E No evidence of work presented.
