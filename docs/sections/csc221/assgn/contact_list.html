ITP 100: Software Design (Python)

Project: Contact List

Due Date

Wednesday, February 29, 2012

What to do:

  1. As with last week, choose a partner. This is a pair programming assignment. Two programmers will work together on one computer and submit one program. You can work with the same partner you did last time, but keep in mind that once we become familiar with this process we will begin mixing things up.

  2. Write a Python program named add_contact.py that asks the user for the following information about a contact:

    1. first name
    2. last name
    3. phone number
    4. email

    The information should be appended to a data file named contact_list.dat.

    A sample session might look something like this: 

    $ python3 add_contact.py 
Please enter your first name: Jeff 
Please enter your last name: Elkner 
Please enter your phone number: 703-228-5771
Please enter your email: jeff@elkner.net
$

    After which, assuming this is the first run of the program, the contact_list.dat file should exist and contain the following: 

    ['Jeff', 'Elkner', '703-228-5771', 'jeff@elkner.net']

    After running it again and entering new information, the file would look something like this: 

    ['Jeff', 'Elkner', '703-228-5771', 'jeff@elkner.net']
['Whitney', 'Singer', '000-000-0000', 'whitneysings@fakemail.com']
  3. Write a program named show_contacts.py that displays an introductory sentence and then the list of contacts in contact_list.dat, one per line. A sample run of the program with the contacts entered in the previous example might look like this: 
    $ show_contacts.py
Contacts:

  Jeffrey Elkner 703-228-5771 jeff@elkner.net
  Whitney Singer 000-000-0000 whitneysings@fakemail.com

Evaluation

You will receive a rubric score between 4 and 0 on this assignment.
