CSC 201 Final Project

Description

The goal of this project is to provide you with a crash course in Python in preparation for your CSC 202: Computer Science II data structures and algorithms course. The plan is to have you reimplement the inventory tracking system you did in lab 4 of the EIMACS AP Computer Science course in Python. By using a familiar problem to learn the new language, we will aim to maximize the transfer of what you learned in your study of Java to Python.

Day 1: The Footwear Class

Download inventory.py and add a body to the Footwear class that will make all the doctests pass. You will need to learn a good deal of Python to be able to complete this otherwise short assignment. I would recommend using Beginning Python Programming for Aspiring Web Developers as the resource to do that, though there are now countless other good resources available on the web.
