Ordinal Numbers

Activity 10.1

Problem

The int t contains an integer between 1 and 50 (inclusive). Write code that outputs the number in words and stores the result in the String inwords. For example, if t is 35 then inwords should contain "thirty five".

Solution

Here is a runnable solution to this problem: 

public class Act10_1 {
    public static void main(String[] args) {
        int t = 29;
        String inwords;
        String[] smallnums = {"narf", "one", "two", "three", "four", "five",
                              "six", "seven", "eight", "nine", "ten",
                              "eleven", "twelve", "thirteen", "fourteen",
                              "fifteen", "sixteen", "seventeen", "eighteen",
                              "nineteen"};
        String[] tens = {"twenty", "thirty", "forty", "fifty"};

        if (t < 20)
            inwords = smallnums[t];
        else if (t % 10 == 0)
            inwords = tens[(t / 10) - 2];
        else
            inwords = tens[(t / 10) - 2] + " " + smallnums[t % 10];

        System.out.println(inwords);
    }
}

Activity 10.2

Problem

The array s of ints contain integers each of which is between 1 and 1000 (inclusive). Write code that stores in the variable ordinals the array of Strings consisting of each number followed by its ordinal number abbreviation, "st", "nd", "rd", or "th". For example if s is the array 

{ 1, 2, 3, 4, 5, 10, 11, 12, 13, 21, 22, 973, 1000 }

then your code should set ordinals to the array 

{ "1st", "2nd", "3rd", "4th", "5th", "10th", "11th", "12th", "13th", "21st", "22nd", "973rd", "1000th" }.

Solution

Given the solution in 10.1 above, you should be able to craft your own solution to this problem, though the following may help you get over the type conversion obstacle: 

public class Act10_2 {
    public static void main(String[] args) {
        int[] s = {21, 22, 973, 1000};
        String[] ordinals = new String[s.length];

        for (int i = 0; i < s.length; i++)
            ordinals[i] = Integer.toString(s[i]);

        for (String ordval: ordinals)
            System.out.println(ordval);
    }
}

The task now is to append the proper ordinal suffix to each string, which you should be able to do on your own.
