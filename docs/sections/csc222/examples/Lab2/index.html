Problem

In this lab you are asked to create your first Java object. Java requires a separate file for each type of object. The name of the file has to match the name of the class. A single file will contain a main method, which most have modifiers public static. This means we will need at least two files, one containing the APPoint class, and one containing the main method.

Let's take a look at how we can put this together to get started with Lab 2. Here is the source for the APPoint class: 

public class APPoint
{
    private double myX;
    private double myY;

    public APPoint( double x, double y )
    {
        myX = x;
        myY = y;
    }

    public double getX() { return myX; }
    public double getY() { return myY; }
    public void setX( double x ) { myX = x; }
    public void setY( double y ) { myY = y; }
}

You have to write this in a file named APPoint.java. Here is the other required file, which I am naming APPointRunner.java: 

public class APPointRunner
{
    public static void main(String[] args)
    {
        APPoint myPoint = new APPoint(4, 5);
        System.out.println("x is " + Double.toString(myPoint.getX()));
    }
}

Now to get this to work from the Unix CLI, you run the following: 

$ javac APPointRunner.java

This will create two new files, APPointRunner.class and APPoint.class, since both files will be compiled. The Java compiler knows to compile APPoint.java because an APPoint instance is created on this line: 

APPoint myPoint = new APPoint(4, 5);

To run the program, type: 

$ java APPointRunner

and you will be greated on the terminal with the text: 

x is 4.0
