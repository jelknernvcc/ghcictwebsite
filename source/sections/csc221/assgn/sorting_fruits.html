ITP 100: Software Design (Python)

Project: Sorting Fruits

Due Date

Monday, February 27, 2012

What to do:

  1. Choose a partner. This is a pair programming assignment. Two programmers will work together on one computer and submit one program.

  2. Write a Python program named get_fruites.py that asks the user for the name of a fruit that they like. Then ask them for nine more. Store these fruits, one per line, in a text file named fruit_list.txt.

    A sample session might look something like this: 

    $ python3 get_fruits.py 
Please enter the name of a fruit you like to eat: cherries
Please enter the name of another fruit you like to eat: blueberries
Please enter the name of another fruit you like to eat: bananas
Please enter the name of another fruit you like to eat: oranges
Please enter the name of another fruit you like to eat: peaches
Please enter the name of another fruit you like to eat: apples
Please enter the name of another fruit you like to eat: strawberries
Please enter the name of another fruit you like to eat: mangos
Please enter the name of another fruit you like to eat: kiwis
Please enter the name of another fruit you like to eat: rasberries
$

    After which, the fruit_list.txt file should exist and contain the following: 

    cherries
blueberries
bananas
oranges
peaches
apples
strawberries
mangos
kiwis
rasberries
  3. Write a program named show_fruits.py that displays an introductory sentence and then the list of fruits in fruit_list.txt enumerated. A sample run of the program with the fruits entered in the previous example might look like this: 
    $ show_fruits.py
Here is a list of the fruits you said you like to eat:

  1. cherries
  2. blueberries
  3. bananas
  4. oranges
  5. peaches
  6. apples
  7. strawberries
  8. mangos
  9. kiwis
  10. rasberries
  4. Write a program named show_sorted_fruits.py that displays an introductory sentence and then the list of fruits in fruit_list.txt enumerated in alphabetical order. A sample run of the program with the fruits entered in the preceding example might look like this: 
    $ show_sorted_fruits.py
Here is a list of the fruits you said you like to eat:

  1. apples
  2. bananas
  3. blueberries
  4. cherries
  5. kiwis
  6. mangos
  7. oranges
  8. peaches
  9. rasberries
  10. strawberries

Evaluation

You will receive a rubric score between 4 and 0 on this assignment.
