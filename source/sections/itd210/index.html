ITD 210

An introduction to web design and development. Stresses a working knowledge of HTML, CSS, and JavaScript. Students will learn to create syntactically valid html 5 web pages using html 5 elements, and to style them with syntactically valid CSS 3, and to add dynamic behavior to them using JavaScript.

This course will combine class discussion, independent and group investigation and practice, and hands-on projects.
