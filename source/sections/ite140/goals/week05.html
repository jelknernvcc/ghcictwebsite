ITE 140: Week 5

Week of Tuesday, September 27, 2022

Overview

This week we will develop our facility with the pandas library and practice using DataFrames to store and manipulate data.

We will begin class this week with each of you sharing what you did for homework exploring pandas.

Goals for Week 5

Using Chapter 5: Data Analysis with pandas as a resource, we will be able to:

  • describe how data is stored in a pandas DataFrame.
  • use the loc method to access parts of a DataFrame.
  • select data from a DataFrame using boolean indexing.
  • set data by replacing values and adding new columns.
  • understand pandas use of NaN and NaT for missing data.
  • use the drop_duplicates method to remove duplicate data.
  • understand the difference between a view and a copy and how to work effectively with each.
