ITE 140: Week 3

Week of Monday, September 12, 2022

Overview

This week we begin Part II. Introduction to pandas in our text. We will be introduced to NumPy and pandas.

Weekend Tasks

Next week we begin working through the programming examples in Chapter 5: Data Analysis with pandas. The first page of the chapter (page 85) starts with two paragraphs filled with vocabulary which I imagine is not familiar to you (it wasn't to me), so our first task will be to develop a definition list containing detailed definitions of the unfamiliar terms.

We will begin class with a discussion on how we should organize our git repos, and then we'll all make our repos conform to the agreed upon organization.

We'll then create a new Markdown file (name and location to be determined in discussion above) for the following vocabulary:

  • Jupyter notebooks
  • pandas
  • interface
  • data sets
  • statistics
  • time series
  • vectorization
  • data alignment
  • cleaning data
  • aggregation
  • descriptive statistics
  • visualization

You'll have the last hour of class and your 90 minutes of homework time to research and write good definitions for the above terms ready to be shared when we come to class on Tuesday.

Tasks for Week 3

  1. Share our datetime programs.
  2. Install and test NumPy and pandas on our computers.
  3. Discuss managing our git repos, including adding a .gitignore file and using git mv and git rm.
  4. Practice the git workflow we will begin using this week.
  5. Make a Python4Excel directory inside your git repo.
  6. Make a Notes directory inside the Python4Excel directory.
  7. Add a file named named NumPyNotes.md inside the Notes directory.
  8. Read Chapter 4: NumPy Foundation in our text, do two things as you do:
    1. Take notes using Markdown in your NumPyNotes.md file, commiting your changes at regular intervals as you go.
    2. Write the examples from the chapter as python programs and add them to a ch04 subdirectory of Python4Excel.

Evaluation

Your grade this week will be equally derived from two things:

  1. Your presentation of you datetime program.
  2. Your regular and frequent commits in your git repo of the example programs and notes from our textbook. I'll give you a grade each time class meets based on your git commits and average them for your weekly grade.
