ITE 170: Linux System Administration

Assignment 15: The Boot Process

Due Date

Friday, May 13th, 2016.

Competencies

Be able to:

  • list system daemons and network services started by the systemd service and socket units.
  • control system daemons and network services using systemctl.
  • describe the boot process.
  • repair file system issues at boot.
  • repair boot loader problems.

What to do:

  1. Read chapter 18 from our text.
  2. Complete the challenge that follows.

Challenge

Build a virtual machine with Centos 7 and KDE as the desktop environment and Lightdm as the display manager. I did this during the past weekend, so I can be of some assistance, but I want to find out what you can do on your own. A few hints would probably be helpful:

  • You'll want to start with a minimal install and then add on to it.
  • You'll need to add an additional repository before you can install Lightdm.
  • You'll need to change the default boot process to get it to start in GUI mode.
